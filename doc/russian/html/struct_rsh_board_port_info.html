Подробное описание

Информация о цифровых портах устройства

Данная структура используется как контейнер для получения информации о цифровых портах.
После вызова метода IRshDevice::Get() с параметром RSH_GET_DEVICE_PORT_INFO, переданная структура будет содержать информацию о цифровых портах данного устройтсва.

Примеры:
LA-1.5PCI_Port.cpp, LA-20USB_Port.cpp, LA-2M5PCI_Port.cpp, LA-2USB_Port.cpp, LA-50USB_Port.cpp, LA-5Net_Port.cpp, LA-i24USB_Port.cpp, LA-n150PCI_Port.cpp, LA-n20-12PCI_Port.cpp, LA48DPCI_Port.cpp, Leonardo2_Port.cpp и Python_Port.cpp.

Конструктор(ы)

RshBoardPortInfo::RshBoardPortInfo ( size_t  maxPortNumber = 32,
size_t  maxConfNumber = 32 
)

Конструктор Можно задать максимальное количество конфигурационных и цифровых портов, которые может вместить структура.
В большинстве случаев в этом нет необходимости.

Данные класса

RSH_BUFFER_PORTINFO RshBoardPortInfo::confs

Конфигурационные порты

Список конфигурационных портов устройства. Используйте поле RshBufferType::Size() для того, чтобы определить количество конфигурационных портов после заполнения структуры.

Прим.
Большинство устройств не имеет конфигурационных портов, только порты данных.
Примеры:
LA-1.5PCI_Port.cpp, LA-20USB_Port.cpp, LA-2M5PCI_Port.cpp, LA-2USB_Port.cpp, LA-50USB_Port.cpp, LA-5Net_Port.cpp, LA-i24USB_Port.cpp, LA-n150PCI_Port.cpp, LA-n20-12PCI_Port.cpp, LA48DPCI_Port.cpp, Leonardo2_Port.cpp и Python_Port.cpp.
RSH_BUFFER_PORTINFO RshBoardPortInfo::ports

Порты данных

Цифровые порты устройства, которые могут быть использованы для ввода и вывода цифровых данных. Используйте поле RshBufferType::Size() для того, чтобы определить количество портов данных после заполнения структуры.

Примеры:
LA-1.5PCI_Port.cpp, LA-20USB_Port.cpp, LA-2M5PCI_Port.cpp, LA-2USB_Port.cpp, LA-50USB_Port.cpp, LA-5Net_Port.cpp, LA-i24USB_Port.cpp, LA-n150PCI_Port.cpp, LA-n20-12PCI_Port.cpp, LA48DPCI_Port.cpp, Leonardo2_Port.cpp и Python_Port.cpp.
Объявления и описания членов структур находятся в файлах:
  • G:/SDK2/HEADERS/Include/RshBoardPortInfo.h
  • G:/SDK2/HEADERS/Include/RshBoardPortInfo.cpp
