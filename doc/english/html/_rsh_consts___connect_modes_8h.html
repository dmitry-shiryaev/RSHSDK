Enumerator
RSH_CONNECT_MODE_BASE 

Connect to device by device index.

If IRshDevice::Connect() is used with this code, method will use device index which was assigned to selected device by the driver.

RSH_CONNECT_MODE_SERIAL_NUMBER 

Connect to device by device serial number.

If IRshDevice::Connect() is used with this code, method will use specified device serial number to connect to device.

RSH_CONNECT_MODE_IP_ADDRESS 

Connect via IP.

This mode is used by some ethernet devices. One can specify ip adress and port in string, for example "192.168.0.5:2034".

RSH_CONNECT_MODE_PLXCHIP_TYPE 

Connect to PLX device.

This mode is used to connect to specific PLX device
Connection string format: №device;PlxChipType (RshDeviceKey::S8*)
example: RshDeviceKey( "0;9054" ) - connects to the first device in the system with a Plx 9054 chip.
