Detailed Description

Direct timers control structure.

This structure can be used for devices with 3 channel timer installed.
Using this structure one can setup each timer work mode and value, programming device to do data acquisition in some sophisticated manner.

See Also
RSH_CAPS_DEVICE_TIMER_8254
Deprecated:
This structure is supposed to use with LA1.5PCI (and possibly LA2USB) device, but all functionality is not supported in current SDK release
