JSC "Rudnev-Shilyaev" device programming manual.

Introduction

JSC "Rudnev-Shilyaev" produces wide variety ADC devices, digital input and output devices, signal generators, oscilloscopes and so on.

All these devices are very different int their parameters. But, nonetheless, there is one simple unificated programming interface, namely IRshDevice. Using this interface one can get access to all features, supported by device in use.

JSC "Rudnev-Shilyaev" software diagram

Software scheme for Windows operating system family is presented on below picture:

Every device is using system driver and one or more abstraction libraries. Top level library implements IRshDevice interface.

One can obtain device control class that implements IRshDevice interface using factory class. Every top level device abstraction library (usually its dll file with name, that corresponds to device name, LA-2USB.dll, for instance) contains static factory with IRshFactory interface, that can be exported from dll using standart Windows API methods (LoadLibrary, GetProcAddress).
To construct concrete control object class with IRshDevice interface one should use IRshFactory::CreateInstance() method.

There is special helper class RshDllClient, that can be used to obtain device interface class with call to only one method - RshDllClient::GetDeviceInterface(). That class also contains some other usefull methods (getting all registerd libraries list, for example).

IRshDevice interface

There is list of all IRshDevice methods. More details on each method can be found on documentation page .
IRshDevice methods:

  • Method IRshDevice::Connect() - connect to device.
  • Method IRshDevice::Init() - set device work parameters.
  • Method IRshDevice::Start() - start data acquisition or generation process.
  • Method IRshDevice::Stop() - stop all activity, cancel any IO operation in process.
  • Method IRshDevice::Get() - get informatin about device, device library or any other device related stuff.
    sometimes this method is used to change internal library or device state also.
  • Method IRshDevice::GetData() - get data buffer from library, or send data buffer to library and into device.
