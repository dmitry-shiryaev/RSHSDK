Analog channels setup example

//every structure derived from RshInitADC has channels list of 32 RshChannel
RshInitMemory initStucture;
...
//You can get acces to specific channel using index (zero based)
initStucture.channels[0].gain = 1; //gain coefficient value
initStucture.channels[0].adjustment = 0.0; // do not use hardware input level adjustment
/*Other channel flags can be set using RshChannel::control field.
All control flags are in RshChannel::Control enum.
Some of those flags have zero value - that are default parameters:
NotUsed - channel won't be used in data acquisiton
NoSynchro - channel won't be used as trigger source
Resist1MOhm - input resist mode is 1MOhm
DC - DC coupling
When RshChannel object is created, it has its control field set to zero,
so default flags are selected.
*/
initStucture.channels[0].control = 0;
/*
This operation is excessive, because its performed in constructor.
But you can always use it to set default control settings.
*/
//alternatively one can set channel default parameters using this one call
initStucture.channels[0].SetDefaultPatameters();
//select channel for data acquisition
initStucture.channels[0].SetUsed();
//alternative call using direct bits manipulation
//initStucture.channels[0].control |= RshChannel::Used;
/*
set input resist to 1MOhm
Excessive operation (and makes no sense because RshChannel::Resist1MOhm == 0), but
it makes code more readable
*/
initStucture.channels[0].control |= RshChannel::Resist1MOhm;
//or one can use class method
//initStucture.channels[0].SetResist1MOhm();
//set input coupling to AC
initStucture.channels[0].control |= RshChannel::AC;
//or one can use class method
//initStucture.channels[0].SetInputAC();
//second channel
initStucture.channels[1].SetUsed();
initStucture.channels[1].SetResist1MOhm(false);
//initStucture.channels[1].SetResist50Ohm(); //its the same
...
//use this channel as trigger source (dont forget to select trigger source in RshInitADC::startType field)
initStucture.channels[1].SetSynchro(true);
//you can copy channel settings from one channel to another
initStucture.channels[2] = initStucture.channels[0];
