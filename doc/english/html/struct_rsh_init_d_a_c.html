Detailed Description

DAC control structure.

This structure is used to get access to device's DAC (or DACs). Some of ADC boards have additional DAC output channels, and using this structure one can set output voltage on these DACs.

Remarks
Devices which whole purpose is signal generation use more complicated structures, such as RshInitGSPF.
See Also
RSH_CAPS_SOFT_INIT_DAC | IRshDevice::Init()
Examples:
RshInitDAC_EN.cpp.

Member Data Documentation

U32 RshInitDAC::id

ID (order number) of DAC.

With this field one can specify, which DAC is used for voltage output.

Remarks
If only one DAC installed, this parameter is ignored.
Device ID is zero based.

RshInitDAC_EN.cpp.

double RshInitDAC::voltage

Output voltage.

Use IRshDevice::Get() method with RSH_GET_DEVICE_OUTPUT_RANGE_VOLTS parameter to find out output voltage range for device.

Remarks
