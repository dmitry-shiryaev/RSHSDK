Detailed Description

Device information structure.

Extended information about device. Contains device base address, serial number, slot number and so on.
One can get list of such structures using IRshDevice::Get() method with RSH_GET_DEVICE_BASE_LIST_EXT parameter.

See Also
RSH_GET_DEVICE_BASE_LIST_EXT
