Detailed Description

Information about digital port.

This structure contains information about digital port.
Usually, one can obtain list of such structures with information about device digital ports using IRshDevice::Get() method with RSH_GET_DEVICE_PORT_INFO parameter.

See Also
RSH_GET_DEVICE_PORT_INFO | RSH_CAPS_SOFT_DIGITAL_PORT_IS_AVAILABLE

Member Data Documentation

U32 RshPortInfo::address

Port address.

Usually just index that can be used to access port, sometimes it is actual address of mapped device register.
One can use addres, obtained by IRshDevice::Get() with RSH_GET_DEVICE_PORT_INFO parameter call, to access digital port via IRshDevice::Init() with RshInitPort structure.
